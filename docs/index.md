# Quantitative Portfolio Engine

Welcome to the **Quantitative Portfolio Engine**, an institutional-grade platform designed for systematic portfolio rebalancing, historical alpha analysis, and risk-managed execution.

## 🚀 Key Features

- **Intelligent Rebalancing**: Automated delta calculation with ISIN-based ticker resolution.
- **Historical Alpha Analysis**: Compare your portfolio's performance against the Quant Strategy and Nifty 50 benchmarks.
- **Security First**: Automated OWASP Top 10 scanning via Bandit and Pip-Audit.
- **Modern Dashboard**: High-performance Streamlit interface for review and allocation.

## 🛠️ Tech Stack

- **Core**: Python 3.13
- **Analysis**: Pandas, NumPy, Scipy
- **UI**: Streamlit, Plotly
- **Security**: Bandit, Pip-Audit
